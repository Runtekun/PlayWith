# PlayWith UIデザインシステム(v1)

## 背景

PlayWithはゲーム友達マッチングアプリ。同ジャンルの既存アプリ「Gamee(gamee.gg)」のビジュアルを参考に、PlayWith独自のUIデザインシステムを定義する。

Gameeの特徴(参考にした点):
- 暖色系のクリーム背景(ライト: `#F4EFE7` / ダーク: `#2A2B2E`)
- パステルピンク・ミントブルー・イエローのアクセントカラー
- 猫耳ヘッドホンをつけたゲーマー女子のkawaii系イラストがブランドイメージ
- 全体的にポップで親しみやすい、カジュアルゲーミング系のトーン

PlayWithもこの方向性(ポップ・親しみやすい・kawaii/カジュアルゲーミング系)を踏襲する。

## カラーパレット

Gamee踏襲(暖色kawaii)案を採用。

| 用途 | 色 | 説明 |
|---|---|---|
| 背景(ベース) | `#F4EFE7` | 暖色クリーム |
| プライマリアクセント | `#F7A6B9` | パステルピンク。ボタン・CTAに使用 |
| セカンダリアクセント | `#8FE3D6` | ミント。カードのグラデーション・タグなどに使用 |
| ターシャリアクセント | `#FFD873` | イエロー。ハイライト・バッジなどに使用 |
| テキスト(基本) | `#333333` | 見出し・本文 |
| テキスト(サブ) | `#666666` / `#888888` | 補足情報 |

検討したが不採用の案:
- B案(薄紫背景+ビビッドなバイオレット/ミント/ピンク): よりクール寄りで独自性はあるが、Gameeとの世界観の近さを優先しA案を採用
- C案(ダーク基調+ネオンアクセント): esports/ガチ勢寄りのクールな印象になるため不採用

## コンポーネントスタイル

角丸大きめの「まるっと丸め(kawaii強め)」案を採用。

- カード: `border-radius: 24px`、押せる感を出すオフセットシャドウ(`box-shadow: 0 6px 0 #e8dcc8`)
- ボタン: `border-radius: 24px`、同様のオフセットシャドウ(`box-shadow: 0 3px 0 <アクセントの濃い色>`)
- プレイヤーカードはTinderライクな縦長比率(画像部分60% + 情報部分40%)を採用

検討したが不採用の案:
- 角丸控えめ+軽い影の中間案: 一度は選ばれたが、最終的に「まるっと丸め」案に変更
- 角ばり+黒フチのポップアート系案: コミック調が強すぎるため不採用

## フォント

丸ゴシック系(M PLUS Rounded 1c / ヒラギノ丸ゴ ProN)を基本フォントとする。

- 見出し: bold
- 本文: 通常ウェイト

検討したが不採用の案:
- 標準ゴシック系: 視認性バランス型だが、kawaii方向の統一感を優先し不採用
- 極太ゴシック(見出し強調): ゲーミング感が強くなりすぎるため不採用

## 主要ボタン文言

スワイプ操作の「参加したい」に相当するボタンは「**一緒にプレイ**」という文言を採用する。

検討したが不採用の案:
- 「気になる」: マッチングアプリ的なニュアンスは出るが、PlayWithはゲームをプレイする目的が明確なので、より直接的な「一緒にプレイ」を採用

## ヘッダー・ボトムナビゲーション

Webアプリだがスマホ利用を主想定とし、フッターは通常のWebサイト的なフッターではなく、**ボトムナビゲーションバー**を採用する。

**ヘッダー**
- 左: サービスロゴ(テキスト「PlayWith」)
- 右: 通知ベル(ラインアイコン、線・図形で構成。絵文字は使わない) + ユーザーアバター(丸型、頭文字またはユーザー画像)

**ボトムナビゲーション**
- 3項目: スワイプ(ホーム) / チャット / マイページ、それぞれアイコン+ラベルを表示
- アイコンはラインアイコン(線・図形で構成、絵文字は使わない)
- 選択中の項目は、アイコンの背景を丸型でプライマリアクセント(`#F7A6B9`)に塗りつぶし、アイコン自体は白抜きにする。ラベルもプライマリアクセント色+bold
- 非選択項目は、背景グレー(`#f0f0f0`)の丸型+グレーのアイコン(`#999999`)

検討したが不採用の案:
- アイコンのみ・ラベルなしの省スペース版: 情報量が少なく分かりにくいため不採用
- 通知ベルを絵文字や塗りつぶし丸アイコンにする案: 絵文字っぽさが好まれず、ラインアイコンに統一

## スコープ

- **ダークモード対応はMVPでは行わない。本リリース(README.mdの「本リリース予定」相当のタイミング)で追加する。**
  - Gameeはライト/ダーク両対応だが、PlayWithは段階的に導入する
  - ダークモードの配色は本リリース着手時に別途検討する
